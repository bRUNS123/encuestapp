# EncuestApp

## Web

See [EncuestApp](https://www.encuestapp.cl/).

## INICIAR PROYECTO

```sh
>cd.. web-app
npm install

>cd.. backend
pip install -r requirements.txt
```


## RUN BACKEND + FRONTEND

```sh
RUN SERVER AND APP:
CONTROL + SHIFT + P
> RUN TASK
> RUN ALL
```
